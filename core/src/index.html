          i
          

            
Что это такое

            

              Обход иногда обманывает провайдера, подсовывая ему поддельный кусочек данных.
              У разных провайдеров эта уловка проходит по-разному: один вариант помогает
              звонкам в Discord, другой — играм. Здесь можно попробовать разные варианты и
              посмотреть, какой сработает лучше именно у вас. Это не сломает программу —
              переключить обратно можно в любой момент.
            

          

        


        

          Эта функция появилась в Zapret 1.10.0. У вашей версии её ещё нет — обновите Zapret
          во вкладке «Обновления», чтобы получить точную настройку под провайдера.

Как запускать при старте

            

              
              
            

          

        


        

          i
          

            
Почему Планировщик, а не автозагрузка Windows

            

              Приложение ставит службу Windows, поэтому ему нужны права администратора. Через обычную
              автозагрузку оно стартовало бы без прав и просило подтверждение UAC при каждом включении
              компьютера. Задача Планировщика выдаёт права сама — молча и один раз при настройке.

Обновление из архива

          
Выберите zip новой версии — ваши списки и настройки перенесутся автоматически.

          
        


        

          i
          

            
Зачем нужны кнопки ниже

            

              IPSet-список — это адреса, которые Zapret обходит напрямую, по IP, а не по
              имени сайта. Часть сервисов блокируют именно по адресу — без свежего списка такие
              сайты обход не подхватит. Кнопка «Обновить» просто скачивает актуальный список с
              сервера разработчиков Zapret.
              


              Файл hosts — системный список Windows, где можно вручную подменить адрес
              сайта. Иногда там случайно оказываются записи, которые мешают открыть YouTube или
              другие сервисы. Кнопка «Проверить» смотрит, не устарел ли ваш файл, и если нужно —
              открывает готовый файл с исправлениями. Сама ничего в системе не меняет — вносить
              правки в hosts всё равно нужно вручную.

Удалить приложение

              
Стирает само приложение Zapret Control (папку, автозагрузку) — удобно перед установкой новой версии.
                Сам обход Zapret и его служба не трогаются и продолжат работать.

            

            
          

        

      


    

  


  
  

    

      
Удалить Zapret Control?

      

        Приложение закроется и его папка будет стёрта — это необратимо. Обход Zapret и его служба
        Windows продолжат работать как есть, это удаляет только саму панель управления.
